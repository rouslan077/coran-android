# 📖 Коран — Читайте и слушайте просто

**Бесплатное** Android-приложение для чтения и прослушивания Корана: арабский текст (классический шрифт или Усмани), транслитерация, переводы на русский, английский и французский языки, аудио-чтение с синхронизацией по аятам. Приложение автоматически использует язык вашего телефона (русский, английский или французский).

## ⬇️ Скачать

**[📥 Скачать APK (последняя версия)](https://github.com/rouslan077/coran-android/releases/latest)**

### Установка
1. Скачайте файл `.apk` на свой Android-телефон.
2. Откройте его и подтвердите установку (разрешите «Установку неизвестных приложений», если потребуется).
3. Откройте приложение и начните чтение.

## ✨ Возможности

- **Полный текст**: все 114 сур с транслитерацией и переводом.
- **Синхронное прослушивание**: аудио следует за текущим аятом — чтецы на русском, английском (Ибрагим Уолк), французском и арабском (Мишари Рашид аль-Афаси).
- **Офлайн-аудио**: скачивайте суры и слушайте без интернета.
- **Продолжение чтения**: возвращайтесь точно туда, где остановились.
- **Избранное и собственные папки**, заметки к аятам.
- **Светлая / тёмная тема**, настройки шрифтов и оформления.
